# 常见的设计模式

## 常见的模式

 * 单体模式
 * 工厂模式
 * 迭代器模式
 * 装饰者模式
 * 策略模式
 * 外观模式
 * 代理模式
 * 中介者模式
 * 观察者模式（订阅发布者模式）
 * 控制反转
 
## 详细说明

其中，最为常用的是 工厂模式、装饰者模式、观察者模式

### 工厂模式

 * 创建或生产一个对象时，包含大量的重复操作
 * 对外的接口简单易懂
 
个人理解，一个脚手架工具就是"工厂模式"，将内部复杂的生产过程封装起来，用户只需要做简单的选择即可

### 装饰者模式

如Express中用到
```javascript
app.use(middlewareA)
app.use(middlewareB)
```
