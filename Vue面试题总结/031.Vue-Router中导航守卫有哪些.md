# Vue-Router中导航守卫有哪些

## 全局守卫

 * `router.beforeEach`
 
 * `router.beforeResolve`
 
 * `router.afterEach`
 
## 路由独享的守卫

 * `beforeEnter`
 
## 组件内的守卫

 * `beforeRouteEnter`
 
 * `beforeRouteUpdate`
 
 * `beforeRouteLeave`
